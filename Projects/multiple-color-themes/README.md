# Multiple color themes project

![Project status](https://img.shields.io/badge/Status-In_Progress-ffea00?style=for-the-badge&logo=linkedin&logoColor=white)

A responsive page that demonstrates advanced CSS theming, grid layouts, and state management using CSS Variables and JavaScript. This project was developed as part of my studing on *Codecademy Front-End Engineer Career Path*.

![Multiple color themes project](./images/demo.gif)

## Key Features
- Seamlessly toggle between Light, Dark, and a special "Custom" theme.
- User theme preference is saved in localStorage, ensuring the choice remains after a page reload.
- Utilizes CSS Grid with `@supports` for a complex, masonry-style gallery that is fully responsive.
- Smooth CSS transitions for colors and backgrounds, plus interactive hover effects using `filter` (`blur` and `hue-rotation`).
- Dropdown menu for theme selection.

## Getting Started (Local Setup)

This project runs entirely in the browser.

1.  **Clone the repository:**
    
    `git clone` [`https://github.com/kolonatalie/frontend-practice-projects.git`](https://github.com/kolonatalie/frontend-practice-projects.git)
    
2.  **Navigate into the project directory:**
    ```bash
    cd frontend-practice-projects/Projects/multiple-color-themes
    ```
    
3.  **Open `index.html`** in your web browser.
